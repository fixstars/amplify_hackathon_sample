# amplify_hackathon_sample

## 実行方法

1. ファイル *sudoku_sample_amplify.py* の、`client.token`のコメントアウトを外してアクセストークンを入力する

2. *sudoku_sample_amplify.py* を実行する

```shell
$ python sudoku_sample_amplify.py
```
